# Quy tắc bắt buộc cho AI Agent

## TRƯỚC KHI LÀM BẤT CỨ ĐIỀU GÌ:
1. Đọc `_docs/PROJECT.md`
2. Đọc `_docs/ARCHITECTURE.md`
3. Đọc `_docs/CONTEXT.md`
4. Đọc `_docs/TASKS.md`

## SAU KHI HOÀN THÀNH MỖI TASK:
1. Cập nhật `_docs/CHANGELOG.md` — ghi lại đã làm gì
2. Cập nhật `_docs/TASKS.md` — chuyển task sang ✅
3. Cập nhật `_docs/CONTEXT.md` — chuẩn bị cho session tiếp theo
4. Nếu phát hiện bug → thêm vào `_docs/BUGS.md`

## KHÔNG ĐƯỢC:
- Sửa file ngoài danh sách trong CONTEXT.md
- Bỏ qua bước đọc docs trước khi làm
- Kết thúc session mà không cập nhật CHANGELOG
